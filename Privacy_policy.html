Privacy Policy
How your data is collected & used
The Basics

Your privacy is important. This policy outlines how personal data is collected, used, and managed from potential and current clients engaging with Nagendra Singh — freelance Data Engineer, Data Scientist & Full Stack Developer.

By using this website or engaging with services offered, you agree to the practices described in this document. This policy is governed by India's Digital Personal Data Protection (DPDP) Act.

1. Data Collection & Usage

Only the minimum data required to operate a service-based business is collected:

  • Contact Data: Names, email addresses, and project details submitted through the contact form — used strictly for professional communication and project management.
  • Project Data: May include credentials for databases, hosting, or technical accounts. The strictest confidentiality is maintained and this data is never shared with third parties without explicit written consent.
  • Usage Data: Basic, anonymised interaction data may be collected to improve website performance. No personal identifiers are retained.
2. Data Security & Storage

For all projects involving Python data analysis or SQL databases, industry-standard security practices are applied — including environment variables, encrypted connections, and secure database tooling — to prevent unauthorized access.

Client data is never stored longer than necessary for the completion of the agreed project scope.

3. Third-Party Services

The following essential services are used. Each processes data according to their own privacy policies:

  • Netlify: Portfolio site hosting and potential project deployment.
  • EmailJS / CallMeBot: Contact form notification delivery.
  • Google Workspace / Slack: Professional client communication.
4. Your Rights (DPDP Act)

Under India's Digital Personal Data Protection Act, you have the right to:

  • Access the personal data held about you at any time.
  • Update / Correct any inaccuracies in your personal data.
  • Request Erasure of your personal information from all records.
  • Withdraw Consent for data processing at any time, without affecting prior lawful processing.

Terms of Service
Governing all professional engagements
1. Services

Freelance web development, Python data analysis, and SQL database management services are offered. All services are defined on a per-project basis via a separate Statement of Work (SOW) agreed upon before work commences.

2. Payment Terms

A non-refundable deposit is required to initiate any project. The remaining balance is due upon project completion and before final code delivery. For larger projects, progress milestones with structured payment schedules will be agreed upon in the SOW.

3. Client Revisions

Each project includes an agreed number of revision rounds during the development phase. Additional revisions beyond this scope are billed separately at an agreed hourly rate. Final code delivery constitutes project acceptance — any modifications thereafter require a new service agreement.

4. Intellectual Property

  • Transfer: Upon full payment, full ownership of the custom-developed solution is transferred to the client, excluding any open-source or third-party libraries used.

By engaging these services, you agree that liability does not extend to any direct, indirect, or consequential business losses — including lost profits or server downtime — arising from the use of delivered work.

No responsibility is assumed for the accuracy or quality of data, content, or credentials provided by the client. It is the client's responsibility to ensure they have the necessary rights to use all provided materials and data.
